# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

- Quiz Taker: A user who wants to browse and take quizzes.
- Quiz Creator: A user who wants to create and manage quizzes.

## Draft User Stories

1. As a Quiz Taker, I want to browse a list of available quizzes so that I can find one that interests me.

2. As a Quiz Taker, I want to filter quizzes by category or quiz type so that I can quickly find quizzes that match my interests.

3. As a Quiz Taker, I want to sort quizzes by title or creation date so that I can browse quizzes more easily.

4. As a Quiz Taker, I want to click on a quiz and view its details so that I can decide whether I want to take it.

5. As a Quiz Taker, I want to answer quiz questions so that I can complete a quiz.

6. As a Quiz Taker, I want to submit my answers so that I can receive a result.

7. As a Quiz Taker, I want to view my quiz result so that I can understand my score or personality type.

8. As a Quiz Creator, I want to create a new quiz so that other users can take it.

9. As a Quiz Creator, I want to edit a quiz so that I can update its title, description, category, or quiz type.

10. As a Quiz Creator, I want to delete a quiz so that I can remove outdated or unwanted quizzes.

11. As a Quiz Creator, I want to add questions and answer choices to a quiz so that the quiz has interactive content.

12. As a Quiz Creator, I want form validation so that incomplete quizzes or questions cannot be saved.

13. As a Quiz Taker, I want to see helpful error messages so that I understand what went wrong if something fails.

14. As a Quiz Taker, I want to be redirected to a result page after submitting a quiz so that I can easily see my outcome.
