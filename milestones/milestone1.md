## Reflection

### 1. What went well during this unit?

Our group was able to agree on a quiz maker app idea that fits the full-stack project requirements well. We identified the main user roles, planned features for quiz creators and quiz takers, and wrote user stories that describe the main user journeys. We also set up the GitHub repository so that all group members can collaborate in the same codebase.

### 2. What were some challenges your group faced in this unit?

One challenge was deciding on an app idea that was interesting but still realistic to build within the project timeline. We also had to think about how to turn a simple quiz maker idea into a full-stack app with enough features, such as creating quizzes, adding questions, taking quizzes, and viewing results. Another challenge was making sure our planned features would match the baseline and custom project requirements.

### 3. What additional support will you need in upcoming units as you continue to work on your final project?

We may need support with designing the database schema for quizzes, questions, answer choices, quiz attempts, and results. We would also benefit from feedback on our wireframes and entity relationship diagram before we begin development. Later, we may need help connecting the React frontend to the Express backend and deploying the app on Render.
